Tentang TKJ

Jaringan Komputer

Perancangan dan konfigurasi LAN, WAN, Router MikroTik, Switch Cisco, Topologi Jaringan, serta Manajemen Bandwidth.

Administrasi Server

Pengelolaan sistem operasi server (Debian/Ubuntu/Windows Server), DNS, Web Server, FTP, dan Virtualisasi.

Hardware & Maintenance

Perakitan PC, troubleshooting kerusakan komponen hardware, instalasi OS, serta perawatan infrastruktur IT.

Aplikasi Perkantoran

Microsoft Word

Pembuatan dokumen formal, laporan teknis, pengerjaan Mail Merge, daftar isi otomatis, serta penyusunan tata letak standar industri.

Microsoft PowerPoint

Merancang presentasi interaktif, desain slide visual bertema IT, animasi transisi menarik, serta penyampaian ide proyek.

Microsoft Excel

Penggunaan rumus dasar & menengah (SUM, AVERAGE, VLOOKUP, IF), tabel data inventaris IT, serta kalkulasi anggaran biaya jaringan.

Keahlian & Konfigurasi Wi-Fi

Konfigurasi Wi-Fi & Router

Pengaturan Access Point, pembatasan kecepatan (Bandwidth Management), pembagian SSID, serta penyettingan IP Gateway.

Keamanan Wi-Fi & Jaringan

Penerapan MAC Address Filtering, penyembunyian SSID (Hide SSID), serta proteksi sistem dari akses tanpa izin.

Pengkabelan Fiber & UTP

Crimping kabel UTP RJ45 (Straight/Cross), penyambungan Fiber Optic (Splicing), serta tes sinyal jaringan.
